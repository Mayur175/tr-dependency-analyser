TR Dependency Analyser — Eclipse Update Site

Eclipse ADT plugin + ABAP backend that detects cross-task and cross-TR object dependencies in SAP transports, recommending a safe DEV → QA → PROD release order.

Install in Eclipse — paste this URL

https://mayur175.github.io/tr-dependency-analyser/

Step 1. In Eclipse / ADT, choose Help → Install New Software…
Step 2. Click Add… and paste the URL above into the Location field. Give it a friendly name (e.g. TR Dependency Analyser) and click Add.
Step 3. Tick TR Analyser for ADT in the list, click Next, accept the licence, click Finish, restart Eclipse.
Step 4. Configure your SAP system in Window → Preferences → TR Analyser, click Test Connection.
